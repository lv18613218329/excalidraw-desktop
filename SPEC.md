# Excalidraw 桌面端教学绘图笔记系统 - 规格说明书

## 1. 项目概述

**项目名称**: Excalidraw-Desktop  
**项目类型**: 跨平台桌面端教学绘图笔记应用  
**核心功能**: 为理科教师（数学、物理、化学）提供高性能的桌面端绘图与课件制作工具  
**目标用户**: 理科教师、课件制作人员、教育机构

## 2. 技术栈

| 层次 | 技术选型 |
|------|----------|
| 外壳框架 | Electron (支持 ARM64/AMD64) |
| 绘图引擎 | Excalidraw (MIT 协议) |
| 前端框架 | React 18+ |
| 状态管理 | Zustand |
| 本地存储 | SQLite / JSON 文件 |
| 构建工具 | Vite + electron-builder |
| 打包目标 | Windows / 统信UOS / 麒麟 |

## 3. 功能模块

### 3.1 基础矢量图形与操作

| 功能 | 描述 |
|------|------|
| 基础几何体 | 直线、矩形、圆形、多边形、箭头 |
| 文本绑定 | 文本与图形绑定显示 |
| 自由旋转 | 中心点/自定义基准点 360° 旋转 |
| 连接点系统 | 图形边缘预设锚点，吸附式箭头连接，拖动自动随动 |
| 图形分割 | 复杂图形打散为子图形，独立填充/旋转 |
| 吸附对齐 | 边缘/中心自动贴合，智能辅助线（水平/垂直/等间距） |

### 3.2 学科专用工具集

#### 3.2.1 物理学科
- **力学**: 滑轮组、弹簧、小车、斜面、受力分析箭头（F1/F2 标签）
- **电路图**: 电阻、电容、电源、开关、电压表/电流表（节点自动吸附）
- **光学**: 凸透镜、凹透镜、光路反射/折射路径（角度预设）

#### 3.2.2 化学学科
- **分子结构**: 六元环、苯环、单双键快速连接
- **实验器材**: 烧瓶、试管、酒精灯、冷凝管
- **液面填充**: 自定义液面高度颜色填充

### 3.3 笔记与存储功能

| 功能 | 描述 |
|------|------|
| 无限画布 | 缩放与无限漫游 |
| 本地文件 | .excalidraw 格式及自定义工程文件 |
| 导出功能 | 高分辨率 PNG、SVG、PDF |

## 4. 非功能需求

### 4.1 系统兼容性
- **Windows**: Win10/11 (x64)
- **国产系统**: 统信 UOS、麒麟 (LoongArch/ARM)

### 4.2 性能要求
- **离线支持**: 完全本地化运行
- **流畅度**: 500+ 图形元素下 60fps 缩放旋转

### 4.3 用户体验
- 符合教师操作习惯的界面布局
- 直观易用的工具栏设计

## 5. 开发路线图

| 阶段 | 任务 |
|------|------|
| Phase 1 | Electron 集成 Excalidraw，打通本地文件读写 |
| Phase 2 | 重写 Library 面板，内置物理/化学矢量库 |
| Phase 3 | 开发图形分割插件，实现逻辑拆解与独立旋转 |
| Phase 4 | UOS/麒麟打包测试，解决权限问题 |

## 6. 技术栈约束规则

> ⚠️ **重要规则**：所有代码实现必须严格遵循需求文档指定的技术栈，禁止使用需求文档未列出的技术。

| 层次 | 指定技术 | 允许 |
|------|----------|------|
| 外壳框架 | Electron | ✅ 仅 Electron |
| 绘图引擎 | Excalidraw | ✅ 仅 Excalidraw |
| 前端框架 | React | ✅ 仅 React |
| 状态管理 | Zustand / Redux | ✅ 二选一 |
| 本地存储 | SQLite / JSON 文件 | ✅ 二选一 |
| 构建工具 | Vite + electron-builder | ✅ |

如需引入其他技术（如其他 UI 框架、状态管理库等），必须先征得用户同意。

## 7. 项目结构

```
Excalidraw/
├── src/
│   ├── main/           # Electron 主进程
│   ├── renderer/       # React 渲染进程
│   │   ├── components/ # UI 组件
│   │   ├── stores/     # Zustand 状态
│   │   ├── libs/       # 学科图形库
│   │   └── utils/      # 工具函数
│   └── preload/        # 预加载脚本
├── resources/          # 静态资源
├── electron-builder.json
├── package.json
└── vite.config.ts
```
